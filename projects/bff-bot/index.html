BFF Bot

Mar 2022

This is an emotional support tool simulating a virtual friendship experience.

Contribution: Web Development
Technologies: JavaScript, p5.js, ML5.js, Rivescript, Figma

Try it out!

I developed this in my Interactive Media Arts class Chatbots for Art's Sake taught by Professor Carrie Wang, as part of a project where I was challenged to create a chatbot that mirrored an aspect of my personality. I had previously been trying to comfort my friends about school-related stress, which motivated me to explore the potential for a machine to provide this kind of emotional support.

I created a desktop-like interface (inspired by Glass Animals Open Source) featuring four applications the user can interact with, all designed to make them feel happy and loved: a chatbot, fun idea generator, photobooth, and an infinite dog scroll.
